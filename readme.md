## hw2: how to run

1. 'hw2' is the package, just drag it in and 'catkin_make'
2. Open 3 terminals


terminal 1: Run the master node
```
roscore
```

terminal 2: Run turtlesim
```
rosmake turtlesim
rosrun turtlesim turtlesim_node
```

terminal 3: Run shape1
```
rosrun hw2 shape1
```
